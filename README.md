# Sistema Gestor de Turnos

Este proyecto es una aplicación web desarrollada en **Flask** que permite gestionar turnos, usuarios y empleados de manera eficiente. El sistema utiliza una base de datos **MySQL** para almacenar la información y cuenta con una interfaz sencilla que facilita el uso de las funcionalidades. El proyecto permite realizar las operaciones de agregar, visualizar, actualizar y eliminar turnos, usuarios y empleados.

## Características Principales

- **Gestión de Turnos**: Permite asignar, visualizar y eliminar turnos para los usuarios y empleados. Los turnos incluyen información como el nombre del usuario, la fecha y la hora del turno.
- **Gestión de Empleados**: Facilita la creación, actualización y eliminación de empleados, quienes serán asignados a los turnos.
- **Gestión de Usuarios**: Permite agregar y administrar los usuarios que necesitan los turnos.
- **Interfaz Web**: El sistema incluye una interfaz web construida con **HTML**, **CSS** (usando **Bootstrap** para facilitar el diseño) y **JavaScript** para interacción dinámica.
- **API REST**: El backend expone rutas para gestionar los turnos, usuarios y empleados de manera que otros sistemas o servicios puedan interactuar con la base de datos.
- **Base de Datos MySQL**: Utiliza una base de datos **MySQL** para almacenar toda la información de los turnos, empleados y usuarios de manera persistente.

## Tecnologías Utilizadas

- **Flask**: Framework de Python para construir la API y la aplicación web.
- **MySQL**: Sistema de gestión de bases de datos relacionales utilizado para almacenar la información.
- **Bootstrap**: Framework de diseño para facilitar la creación de una interfaz web atractiva y responsiva.
- **JavaScript**: Lenguaje de programación para agregar interactividad a las páginas web.
- **Python**: Lenguaje de programación utilizado para desarrollar la lógica del backend.
